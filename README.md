# Lex Programs

## Executing

```
    git clone https://github.com/shb9019/Lex-Programs.git
    cd Lex-Programs
    # Make sure you have flex ('sudo apt install flex')
    lex <filename>.l
    g++ --std=c++14 lex.yy.c
    ./a.out
```
